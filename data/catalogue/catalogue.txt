But du jeu : 
	L'objectif est de relié entre elles des îles grâce à des ponts. Il faut obtenir à la fin un seul bloque qui relie toutes les îles.

Rêgles du jeu : 

	- Un pont commence et se termine sur une île
	- Deux îles ont au maximum deux ponts en commun
	- deux ponts ne peuvent pas se croiser
	- Les ponts sont soit verticales, soit horizontales
	- La valeur sur les îles est le nombre de ponts qui passent pas cette île
	
	Le partie se termine quand toutes les îles sont reliées et qu'elles possèdent tous leurs ponts.
	
Aides : 

	- Coups possibles : indique les ponts que vous pouvez faire à partir d'une île en survolant cette dernière
	- Détection d'erreurs : Permet de supprimer les mauvais ponts que vous avez créé
	- Suggestion de coups : Permet d'obtenir un coup à jouer en fonction de la situation dans laquelle vous êtes, si votre grille possède des erreurs, alors elles vous seront indiqués et vous devrez les corriger avant d'avoir un coup à jouer
	- Auto-correcteur : Permet de supprimer les mauvais ponts dès leur création création. Cette aide est désactivée par défaut.

Mode de jeu : 

	- Tutoriel : permet d'apprendre les bases du hashi grâce à un exemple ou on vous accompagne dans la partie
	- Normal : Mode de jeu simple, dans lequel un crhonomètre est lancé mais sans limite de temps.
	
			Les aides suivantes sont activées : 
				- Coups possibles
				- Suggestion de coups
				
		  	 Les aides vuivantes sont accessibles : 
			   	- Détection d'erreurs
			   	- Auto-correcteurs
			   	
	- Contre la montre :  Mode de jeu ou vous devrez terminer le plateau avant le temps imparti.
	
			Les aides suivantes sont activables : 
				- Coups possibles
				- Suggestion de coups
				
		  	Les aides vuivantes sont accessibles : 
			   	- Détection d'erreurs
			   	- Auto-correcteurs
			   	
			l'acrivation d'une aide entraine une baisse du chrono.
			
			
	- Génie : Mode de jeu dans lequel toutes les aides sont désactivées.
	
	
Difficultés : 

	- Facile : les plateaux sont petits et sans grande difficultées
	- Moyen : les plateaux ont une taille variables sans grande difficultées 
	- difficile : les plateaux sont grand et ont une difficulté élevé
	
	

Tactics : 

	- île avec un simple voisin : Si une île possède un unique voisin
