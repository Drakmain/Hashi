    class Normal

    La classe Normal est une sous-classe de la classe ContreLaMontre, elle possède les mêmes caractèristiques. La seule chose qui change est le sens du chronomètre

    Elle peut donc faire tous ce que peut faire la classe ContreLaMontre.

def lancerChrono(unLabel)
  if(@chrono == 0)then
    @chrono = Chrono.new(unLabel)
    @chrono.lancerChrono
  else
    valeur = @chrono
    @chrono = Chrono.new(unLabel)
    @chrono.lancerChronoValeur(valeur)
  end
end
